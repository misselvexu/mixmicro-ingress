== RateLimiter Config Properties

- 网关限流

[source,json]
----

[
  {
    "resource": "ws",
    "count": 1000
  },
  {
    "resource": "/demo/echo",
    "limitApp": "default",
    "grade": 1,
    "count": 5,
    "strategy": 0,
    "controlBehavior": 0,
    "clusterMode": false
  }
]

----

- 网关服务熔断降级配置

[source,json]
----

[
  {
    "resource": "workshop",
    "limitApp": "default",
    "count": 500.0,
    "timeWindow": 10,
    "grade": 0,
    "rtSlowRequestAmount": 5,
    "minRequestAmount": 5
  }
]

----

- 网关黑白名单

[source,json]
----

[
  {
    "resource": "workshop",
    "limitApp": "app1,app2",
    "strategy": 0
  }
]

----
